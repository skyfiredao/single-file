## Projects

### [ai_translate](ai_translate/) — Private Offline Translator

A single-file translator that connects to your local LLM for private, offline text translation. For anyone who wants AI translation without sending text to the cloud.

### [cat_doc](cat_doc/) — Browser-Based Document Workspace

A single-file HTML file that lets you browse, edit, and create Markdown and code files directly in the browser. No server, no install, works offline. For anyone who wants a lightweight document workspace they can drop anywhere.

### [cat_git](cat_git/) — Browser-Based Git Viewer

A single-file HTML file that lets you browse commit history, view side-by-side diffs, and checkout file versions. No install needed. For quick Git repository inspection without setting up any tools.

### [prompt_gen](prompt_gen/) — Prompt Optimizer

A single-file tool that optimizes your LLM prompts using a local model. Write a prompt, pick a model, get an improved version streamed back. For prompt engineers who iterate on prompts without cloud dependencies.

### [ps0](ps0/) — Self-Hosted Service Portal

A single-file site aggregation panel. Edit a Markdown file to organize all your self-hosted services into a clean portal page. For anyone running multiple services who needs one starting page to reach them all.

### [skychat](skychat/) — Encrypted Web Chat

End-to-end encrypted chat where the server never sees plaintext. For private communication on your own infrastructure.
