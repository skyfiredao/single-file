## 项目

### [ai_translate](ai_translate/) — 私密离线翻译

单文件翻译器，连接本地 LLM 实现私密离线翻译。适合不想把文本发到云端的翻译需求。

### [cat_doc](cat_doc/) — 浏览器文档工作空间

单个 HTML 文件，直接在浏览器中浏览、编辑和创建 Markdown 及代码文件，无需服务器，无需安装，支持离线。适合需要一个轻量、随处可放的文档工作空间。

### [cat_git](cat_git/) — 浏览器 Git 查看器

单个 HTML 文件，浏览提交历史、查看并排 diff、签出文件版本，无需安装任何工具。适合快速查看 Git 仓库的场景。

### [prompt_gen](prompt_gen/) — 提示词优化器

单文件提示词优化工具，用本地 LLM 改进你的提示词。写一个提示词，选一个模型，实时流式返回优化版本。适合不依赖云服务、在本地迭代提示词的工程师。

### [ps0](ps0/) — 自托管服务门户

单文件站点聚合面板。编辑一个 Markdown 文件，把所有自托管服务整理成一个干净的门户页面。适合运行多个服务、需要一个统一起始页的人。

### [skychat](skychat/) — 加密 Web 聊天

端到端加密聊天，服务器永远看不到明文。适合在自有基础设施上进行私密通信。
